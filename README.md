# 🚀 EC2 DevOps Dashboard

This project demonstrates how to deploy a static web application on AWS EC2 using Apache Web Server.

## 📌 Project Overview

A simple and responsive DevOps-style dashboard that displays system-like metrics such as CPU usage, RAM usage, IP address, and live time.

## 🛠️ Technologies Used

* AWS EC2
* Apache Web Server (httpd)
* HTML, CSS, JavaScript
* GitHub (for version control)

## ⚙️ Features

* Dynamic dashboard UI
* Auto-updating system metrics (CPU, RAM)
* Live server time display
* Responsive layout
* Cloud deployment using EC2

## 🚀 Deployment Steps

1. Launch EC2 instance (Amazon Linux)
2. Install Apache server
3. Upload project files
4. Start httpd service
5. Access via Public IP
